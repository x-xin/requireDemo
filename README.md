##### 目录结构

    / 根目录
    |__ dest                 发行目录
    |__ dev                  开发目录
    |   |__ js
            |__ app          模块
                |__ a.js     模块
                |__ b.js
                |__ main.js  入口文件 
            |__ lib          资源
            |__ config.js    配置文件
    |__ package.json         依赖管理
    |__ gulpfile.js          任务文件
    |__ index.html           html
    
    
#####　运行方式

###### 安装

    npm install

###### 编译

    gulp r or gulp
